# إنشاء أول مستودع لك

لقد قمت بتثبيت تطبيق سطح المكتب GitHub وقمت بربطه بحسابك. الآن حان الوقت للبدء في وضع شيء تحت التحكم بالنسخ!

في تطبيق سطح المكتب GitHub، انقر على السهم زائد في الزاوية العلوية اليسرى. افتراضيًا، يتيح لك هذا **إنشاء** مستودع جديد. (يمكنك أيضًا **إضافة** مستودع موجود بالفعل على نظامك الذي لا يعرف عنه التطبيق بعد، أو **استنساخ** مستودع موجود بالفعل على GitHub ولكن ليس على جهاز الكمبيوتر الخاص بك.) قم بإعطاء المستودع اسمًا وصفيًا (بدون مسافات!) واختر مكانًا لحفظه. افتراضيًا، يحفظ المستودعات الجديدة في `C:\Users\[[اسم المستخدم]]\Documents\GitHub\[[اسم المستودع]]`، وهو نفس الشيء مثل قول `مستنداتي > GitHub > [[اسم المستودع]]`. يمكنك اختيار موقع مختلف إذا كنت تفضل ذلك. بمجرد اختيارك لاسم ومكان لحفظ المستودع، قم بالنقر على **إنشاء المستودع**.

## إنشاء المستودع
![إنشاء مستودع جديد!](/images/create-repo.png)

لقد أنشأت مجلدًا لتتبع عملك بهذا المشروع!

عند إنشاء مستودعك الأول، سيقوم تلقائيًا بإجراء أول تأكيد لك. يحتوي هذا التأكيد الأول على ملفات تساعد git في تتبع عملك بشكل صحيح. يمكنك تجاهل هذه الملفات في الوقت الحالي، لكن لا تقم بنقلها أو حذفها! هذه الملفات عادة ما تكون مخفية، لذا من المحتمل أن تراها فقط إذا قمت بتمكين عرض الملفات النظامية المخفية على جهاز الكمبيوتر الخاص بك.
